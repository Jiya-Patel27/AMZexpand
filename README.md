# AMZexpand Landing Page Recreation

## Overview

This project is a recreation of the AMZexpand landing page, developed as part of the Frontend Developer role assignment for DataVinci. It is fully responsive, accessible, and follows modern frontend development best practices.

## Links

- **Live Preview**: [https://amz-expand-psi.vercel.app/](https://amz-expand-psi.vercel.app/)
- **Source Code**: [https://github.com/Jiya-Patel27/AMZexpand](https://github.com/Jiya-Patel27/AMZexpand)

## Technical Stack

- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Custom styling with CSS variables, grid, flexbox, and responsive media queries
- **JavaScript**: Basic event handling and DOM manipulation
- **No external libraries/frameworks**: Pure HTML/CSS/JS for maximum performance and simplicity

### Frontend Best Practices Followed

- **Semantic HTML5**: Meaningful tags (header, section, nav, etc.) for accessibility and SEO
- **CSS Variables**: Reusable color, spacing, and typography tokens
- **Responsive Design**: Mobile‑first approach with appropriate media queries
- **Clean Code Structure**: Well‑organized CSS with commented section headers
- **Performance Focus**: No heavy frameworks, minimal code footprint
- **Accessibility**: Semantic structure, appropriate contrast ratios, and interactive states
